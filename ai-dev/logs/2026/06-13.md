- `TestDefaultAgentEngineCancel.java`: 8 tests вҖ” getSessionStatus fail-fast for non-existent, getSessionStatus pending for newly created, getSessionStatus running during execution + cancelled after graceful cancel (lifecycle), graceful cancel fail-fast on non-existent, idle-session cancel, end-to-end graceful cancel (tool blocks via latch вҶ’ cancel вҶ’ loop stops at boundary вҶ’ cancelled result + session status), end-to-end forced cancel (LLM call blocks вҶ’ forced interrupt вҶ’ cancelled result, terminates < 30s), event publication (SESSION_CANCEL_REQUESTED + SESSION_CANCELLED emitted, EXECUTION_COMPLETED NOT emitted)
