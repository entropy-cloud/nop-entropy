# AI 开发日志 — 2026-05-02

### 2026-05-02

- 完成 `nop-code` 多语言代码索引设计文档：`ai-dev/design/nop-code/language-agnostic-code-index-design.md`（940+ 行）
- 设计目标：将 `nop-code` 从 Java 专用代码索引重构为支持 Java/Python/TypeScript 的多语言索引服务
- 核心决策：

  4. 语言特有字段用 `extData` JSON 扩展，避免类爆炸
  5. Python/TypeScript 解析用 tree-sitter（`io.github.bonede:tree-sitter`），正则降级
