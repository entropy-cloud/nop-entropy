## 快速开始

如果您是第一次使用OpenSpec，建议按照以下步骤：

1. 阅读完整的[OpenSpec安装指南](openspec-installation.md)
2. 按照安装指南完成OpenSpec的安装
3. 使用OpenCode的斜杠命令创建第一个变更提案
4. 查看[AGENTS.md](../../openspec/AGENTS.md)了解OpenSpec工作流

## AI工具集成

Nop平台支持与多种AI编码助手集成，包括：

- **OpenCode**：主要推荐的AI编码工具
- **Claude Code**：Anthropic的AI编码助手
- **Cursor**：AI驱动的代码编辑器
- **GitHub Copilot**：GitHub的AI编程助手

OpenSpec可以与所有这些工具无缝集成，提供统一的规范驱动开发工作流。
