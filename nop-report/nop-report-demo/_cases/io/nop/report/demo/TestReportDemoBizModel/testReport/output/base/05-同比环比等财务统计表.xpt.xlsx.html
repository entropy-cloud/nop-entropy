同比环比等财务统计表说明：
FR中使用层次坐标公式可以轻松实现环比、同期比、累计、排名等财务统计表
1、环比
该表中销售额是按照月份汇总，并按照年份分组。环比指的是当月销售额与上月对比变化比例。
2、同比
同比指的是当月数据与上年同期数据的变化比例。
3、占比
当前月在该年中所占比例
4、排名
如该表中在每一年内各月销售额排名及所有年中各月销售额排名
5、累计
如该表中同一年内每个月的销售额依次累计及所有年所有月的销售额累计
