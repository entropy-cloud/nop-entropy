# 通用文档简化提示词

## 任务目标

对现有技术文档进行系统化精简和优化，确保文档内容：
1. **消除重复**：识别并合并重复的内容
2. **结构优化**：建立清晰的文档层次结构
3. **内容聚焦**：保留核心信息，移除冗余内容
4. **可维护性**：建立易于维护的文档体系

## 核心原则

### 1. 内容精简原则
- **避免重复**：相同概念在不同文档中只保留一份完整描述
- **聚焦核心**：只保留最关键的功能和特性描述
- **移除过时**：删除不再使用或已废弃的功能说明
- **合并相似**：将功能相似的文档合并为统一文档

### 2. 结构优化原则
- **层次清晰**：建立清晰的文档层次结构
- **模块化组织**：按功能模块组织文档内容
- **统一入口**：提供统一的文档索引和导航
- **交叉引用**：建立合理的文档间引用关系

### 3. 可读性原则
- **简洁明了**：使用简洁的语言表达复杂概念
- **示例精炼**：提供精炼但完整的代码示例
- **重点突出**：突出显示关键信息和最佳实践
- **术语统一**：使用统一的术语体系

## 简化流程

### 阶段1：文档分析
1. **内容扫描**：扫描所有相关文档
2. **重复识别**：识别重复的概念和内容
3. **结构分析**：分析文档的组织结构
4. **依赖分析**：分析文档间的依赖关系

### 阶段2：内容整合
1. **核心文档创建**：创建统一的核心概念文档
2. **重复内容合并**：将重复内容合并到核心文档
3. **专题文档重组**：按功能专题重组相关文档
4. **示例统一**：统一和优化代码示例

### 阶段3：结构优化
1. **层次重构**：重构文档的层次结构
2. **导航优化**：优化文档导航和索引
3. **交叉引用**：建立合理的交叉引用机制
4. **入口统一**：提供统一的文档入口点

### 阶段4：质量验证
1. **完整性验证**：确保所有重要内容都得到保留
2. **一致性验证**：验证术语和示例的一致性
3. **可读性验证**：验证文档的可读性和易理解性
4. **维护性验证**：验证文档体系的易维护性

## 具体实施策略

### 1. 重复内容处理策略
- **概念重复**：相同概念在不同文档中只保留一份完整描述
- **示例重复**：相似的代码示例合并为通用示例
- **配置重复**：相同的配置说明合并为统一配置指南
- **最佳实践重复**：相同的最佳实践合并为统一指南

### 2. 文档结构优化策略
- **核心文档**：创建统一的核心概念文档
- **专题文档**：按功能专题组织相关文档
- **参考文档**：创建快速参考文档
- **教程文档**：创建循序渐进的教程文档

### 3. 内容精简策略
- **移除过时内容**：删除不再使用的功能说明
- **合并相似功能**：将功能相似的说明合并
- **简化复杂描述**：将复杂描述简化为易于理解的形式
- **聚焦核心价值**：只保留最有价值的内容

## 成功指标

### 量化指标
- **文档数量减少**：目标减少30-50%的文档数量
- **内容重复率降低**：目标将重复内容减少80%以上
- **平均文档长度优化**：目标将平均文档长度控制在合理范围内
- **导航效率提升**：目标减少50%的导航步骤

### 质量指标
- **内容完整性**：确保所有重要功能都有完整说明
- **结构清晰度**：文档结构清晰，易于理解和导航
- **维护便利性**：文档体系易于维护和更新
- **用户满意度**：用户能够快速找到所需信息

## 风险控制

### 潜在风险
1. **信息丢失风险**：过度精简可能导致重要信息丢失
2. **结构混乱风险**：结构优化不当可能导致导航困难
3. **维护困难风险**：过于复杂的文档结构可能难以维护
4. **用户适应风险**：用户需要时间适应新的文档结构

### 风险缓解措施
1. **渐进式优化**：采用渐进式的优化策略
2. **用户反馈**：收集用户反馈并持续改进
3. **备份机制**：保留原始文档作为备份
4. **测试验证**：通过测试验证优化效果

## 实施建议

### 优先级排序
1. **高优先级**：处理严重重复和结构混乱的文档
2. **中优先级**：优化内容组织和导航结构
3. **低优先级**：细节优化和美化工作

### 分阶段实施
1. **第一阶段**：分析现状和制定方案
2. **第二阶段**：核心文档创建和内容整合
3. **第三阶段**：结构优化和导航改进
4. **第四阶段**：质量验证和持续优化

### 持续改进
- 建立文档质量监控机制
- 定期收集用户反馈
- 持续优化文档内容和结构
- 保持文档的时效性和准确性
